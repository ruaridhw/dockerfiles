# r-pkgdev

A Docker container providing a base image for R package development
using the [rocker/r-ver](https://hub.docker.com/r/rocker/r-ver/) Docker
image with devtools.

Aims to provide an intermediatary layer between base R and the other
[rocker images](https://hub.docker.com/r/rocker/) which contain a full
RStudio / Tidyverse / publishing stack.
